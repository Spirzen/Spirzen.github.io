Архитектура веб-энциклопедии

Сайт it-knowledge-base — read-only SSG без бэкенда и авторизации: Markdown в Git → сборка Docusaurus → статический build/ на GitHub Pages (spirzen.ru).

  1. Контекст (C4) Автор пушит в репозиторий → GitHub Actions (deploy.yml) → npm ci + build → CDN GitHub Pages с CNAME.
  2. Репозиторий docs/ (~3000 MDX), src/ (swizzled theme, 150+ React-компонентов), sidebars.js, static/.
  3. Сборка Препроцессоры (build-init-links.mjs, build-doc-search-index.mjs, build-doc-redirects.mjs) → Docusaurus preset-classic, Remark/Rehype, lunr-search, redirects, image-zoom.
  4. Деплой Node 20 в CI, peaceiris/actions-gh-pages → ветка gh-pages, Yandex Metrica на проде.
  5. Runtime Гидрация React: Navbar + DocSearch, sidebar, MDX-страница с TOC и «См. также», клиентский поиск по doc-search-index.json, интерактивные демо-блоки.
